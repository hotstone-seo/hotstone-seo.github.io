# HotStone SEO Github Page

## Get Started

1. Install Ruby
    ```bash
    brew install ruby
    ```
2. Install Jekyll
    ```bash
    gem install bundler jekyll
    ```
3. Start the website
    ```bash
    bundle exec jekyll start
    ```
