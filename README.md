# Hot-Stone SEO Github Page

## Get Started

1. Install Ruby

    ```bash
    brew install ruby
    ```

2. Install Jekyll

    ```bash
    gem install bundler jekyll
    ```

3. Install gems declared in `Gemfile`

    ```bash
    bundle install
    ```

4. Start the website

    ```bash
    bundle exec jekyll serve
    ```
